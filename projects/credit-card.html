Active vs Churned Customer Behavior Analysis

GitHub Repo View Dashboard

Problem & Approach

Customer churn is a critical problem in the banking industry. Understanding behavioral differences between active and churned customers can help improve retention strategies.

This project analyzes transaction behavior using exploratory data analysis, descriptive statistics, and inferential statistical testing to identify meaningful differences.

Analysis Approach

  • Exploratory Data Analysis (EDA) using Python
  • Descriptive statistics comparison between customer groups
  • Inferential statistical testing (two-sample t-test)
  • Interactive dashboard visualization using Tableau

Interactive Dashboard

Interactive dashboard comparing transaction behavior between active and churned customers.

Key Insights

  • Active customers have significantly higher transaction amounts than churned customers
  • Distribution analysis shows clear behavioral separation between groups
  • Statistical testing confirms the difference is significant
  • Transaction amount is a strong indicator of customer loyalty
