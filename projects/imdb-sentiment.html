IMDb Sentiment Analysis

Deep Learning Model for Movie Review Sentiment Classification

GitHub Repo Kaggle Notebook Live Demo

Problem & Approach

Movie review platforms like IMDb generate large volumes of user feedback in textual form. Manually analyzing these reviews is inefficient, especially when identifying negative sentiment or overall audience perception.

This project builds a Natural Language Processing (NLP) model to automatically classify reviews into positive or negative sentiment using a Bidirectional LSTM architecture.

The workflow includes text preprocessing, feature engineering, baseline model development, and improvement using Word2Vec embeddings with transfer learning.

Modeling Approach

  • Built baseline ANN models using TextVectorization and Embedding
  • Observed overfitting behavior in training-from-scratch approach
  • Implemented Word2Vec embeddings to improve representation learning
  • Applied EarlyStopping to reduce overfitting and stabilize training
  • Final model: Bidirectional LSTM with pretrained embedding

Deployment

The final model is deployed on Hugging Face Spaces, allowing users to input custom movie reviews and receive real-time sentiment predictions.

Results

  • Achieved ~88% accuracy on test dataset
  • Reduced overfitting compared to baseline models
  • Improved generalization using Word2Vec embeddings
  • Model captures sentiment patterns but still challenged by sarcasm and ambiguity
